Adımlar:

1)  Aşağıdaki bağlantıdan tesseract ocr'ın işletim sisteminize uygun sürümünü (32-bit ya da 64-bit) indirin ve bilgisayarınıza kurun.
	https://github.com/UB-Mannheim/tesseract/wiki
2)  Kurulum esnasındaki yükleme dizinini (ör:C:\Program Files\Tesseract-OCR) bir yere kopyalayın.
3)  Herhangi bir IDE'de (Pycharm, vb.) yeni bir Python projesi oluşturun.
4)  Komut satırını (cmd) çalıştırın ve komut satırında açmış olduğunuz Python projenizin dizinine gidin.
5)  Komut satırına aşağıdakini yazarak gerekli olan kütüphaneyi projenize yükleyin.
	pip install pytesseract
6)  GitHub üzerinden erişebildiğiniz main.py dosyasını bilgisayarınıza indirin ve proje dizininin içerisine taşıyın.
7)  IDE'nizde main.py dosyasını açın ve tesseract.exe dosyasının yolunu daha önce kurulum sırasında kopyalamış olduğunuz yükleme dizininin adresiyle değiştirin.
8)  main.py dosyasını kaydedin.
9)  Üzerindeki yazıyı okutmak istediğiniz resim dosyasını img1.png olarak proje dizininin içerisine koyun.
10) main.py dosyasını çalıştırın.
